IMDb Gallery October 2017

The Art of Web Programming, Computer Science Department, UIUC



tools: React.js, SCSS, Webpack, Babel, The Movie Database


My first project using React.js! This is a single-page React app that uses movie data from The Movie Database API. I implemented the following user interactions:


  • Search for movies with live-filtered results
  • Sort query results by popularity and vote count
  • Sort query results in ascending or descending order
  • View films in gallery view and filter by film genre
  • Select a movie to reveal additional details, such as synopsis and average rating

Check it out on Github!

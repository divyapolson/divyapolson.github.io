My first project using React.js! This is a single-page React app that interacts with movie data from The Movie Database. Users can search for movies and sort query results by popularity and vote count (ascending or descending). The app also features a gallery view, where users can filter by film genre. Selecting a movie reveals additional details such as the synopsis and average rating.
